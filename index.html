Power BI

  1. CSV / XLSX: Home → Get data → pick the file → Load.
  2. Star schema (ZIP): load each CSV, then relate them on DrugIDPK — slice by every field with no row blow-up.
  3. Mark AuthorisedDateISO as a Date column for time intelligence.

Amazon QuickSight

  1. CSV or NDJSON upload best — both are flat and rectangular.
  2. New dataset → Upload a file (or S3) → confirm field types.
  3. Use Explode by Active Substance for substance-level analysis.
