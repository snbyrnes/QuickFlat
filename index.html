QuickFlat

HPRA medicines XML → flat CSV / JSON / NDJSON for Power BI & QuickSight

View on GitHub

Drop the HPRA human medicines XML list below and QuickFlat turns it into a clean, rectangular table you can load straight into Power BI or Amazon QuickSight. It recognises the HPRA schema (and falls back to generic flattening for any other XML). Everything runs in your browser — no file is ever uploaded.

Loading the output

Power BI

  1. CSV: Home → Get data → Text/CSV, pick the file, Load.
  2. JSON: Get data → JSON → To Table → expand columns.
  3. Set AuthorisedDateISO as a Date column for time intelligence.

Amazon QuickSight

  1. CSV or NDJSON upload best — both are flat and rectangular.
  2. New dataset → Upload a file (or S3) → confirm field types.
  3. Use Explode by Active Substance for substance-level analysis.
